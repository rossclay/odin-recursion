Inspect elements and interact with console to use functions.

fibs(n):

takes a number, and returns an array containing that many numbers of the Fibonacci sequence, using a for loop.

fibsRec(n):

takes a number, and returns an array containing that many numbers of the Fibonacci sequence, using Recursion.

mergeSort(arr):

takes an array, and returns a sorted array, using a recursive merge srot methodology.
